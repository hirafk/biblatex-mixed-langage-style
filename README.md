# Bilingual BibLaTeX Style (JP/EN)

英語と日本語の文献が混在する論文・レポートに特化した BibLaTeX スタイルファイルです。

## 概要

SIST02, ISO690, NLM, Vancouver などの主要な引用規格を参考にしつつ、**自然科学系の英語論文で標準的なスタイル**をベースに調整しています。
「英語文献のスタイルはそのままに、日本語文献もそれに調和する形で違和感なく出力したい」というニーズに応えるために作成しました。

## 特徴

* **言語の自動判定**: 英語文献 (`langid = english`) と日本語文献でフォーマットを自動的に切り替えます。
* **著者名の処理**: 英語は名前をイニシャル、日本語は姓名を表示します。

## インストール方法

以下の **A** または **B** いずれかの方法で配置してください。

### A. プロジェクトごとに使う（簡単）
`.sty` ファイルを、執筆中の `.tex` ファイルと同じディレクトリ（フォルダ）に置いてください。

### B. 全てのプロジェクトで使う（推奨）
TeX の個人用ツリー（texmfツリー）に配置することで、パスを指定せずにどこからでも読み込めるようになります。

**Windows の場合:**
`C:\Users\ユーザー名\texmf\tex\latex\任意のフォルダ名\`
の中に `.sty` ファイルを配置してください。

**macOS の場合:**
`~/Library/texmf/tex/latex/任意のフォルダ名/`
の中に `.sty` ファイルを配置してください。

※ 配置後、ターミナルで `kpsewhich ファイル名.sty` を実行し、パスが表示されれば認識されています。

## 使い方

プリアンブル（`\begin{document}` の前）でパッケージとして読み込みます。
BibLaTeX の設定はパッケージ内に含まれているため、別途 `\usepackage{biblatex}` を記述する必要はありません。

```latex
% パッケージの読み込み
\usepackage{あなたのファイル名} 

% 文献データベース(.bib)の指定
\addbibresource{references.bib}

\begin{document}
...
```
